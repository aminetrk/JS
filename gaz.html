Functions and Gas Average Project

This page answers the questions about functions and runs a JavaScript program that calculates the average weekly gas bill.

Questions

1. What is the purpose of a function?

A function is a reusable block of code that performs a specific task. It lets us organize code, avoid repeating ourselves, and make programs easier to read and maintain.

2. Loop Question

Code:

for (let i = 1; i <= 5; i++) {
    console.log("Snack #" + i + " given to customer");
}

Question: Using the code above, how many times does the loop execute?

Answer: The loop executes 5 times.
