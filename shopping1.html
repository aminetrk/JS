Questions

  1. Name a situation where you would need to create an "if" statement. (There are many answers to this question.)
  2. When comparing values, in what data type is the result stored? (number, string, or Boolean?)

Answers

  1. Using if statements is essential for validating user input or checking correctness. For example, if a user tries to enter an invalid name when you're asking for a user's name, JavaScript can catch the error and prevent invalid input.
  2. When comparing values, the data type returned is a Boolean value (true or false).
